Instructions

  1. In a terminal on your computer, create an SSH tunnel between the compute node and your computer using the following command] 
    ssh {{username}}@{{machine_name}} -L {{port}}:{{hostname}}:{{port}}
  2. Start ParaView on your computer
  3. Go to File -> Connect
  4. Click on Add Server
  5. Click on Configure, select Manual, and enter: 
    name = {{machine_name}}
                    type = Client/Server
                    host = {{machine_name}}
                    port = {{port}}
  6. Click on Save
  7. Select the server from the list and click on Connect. If you refresh this page, you should see "Client connected" in the command-line output section.
  8. Open a file in ParaView (it will point to your remote filesystem) and visualize it as usual.

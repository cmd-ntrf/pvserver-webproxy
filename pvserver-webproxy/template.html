Instructions

  1. In a terminal on your computer, create an SSH tunnel between the compute node and your computer using the following command: 
    ssh {{username}}@{{machine_name}} -L 11111:{{hostname}}:{{port}}
  2. Start ParaView {{version}} on your computer
  3. Go to File -> Connect
  4. Click on Add Server
  5. Enter:
    • Name: {{machine_name}}
    • Server Type: Client/Server
    • Host: localhost
    • Port: 11111
  6. Click on Configure, select Manual
  7. Click on Save
  8. Select the server from the list and click on Connect.
  9. If you refresh this page, you should see "Client connected" in the command-line output section.
  10. Open a file in ParaView (it will point to your remote filesystem) and visualize it as usual.
